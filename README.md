# dinsospsar.github.io
Web ini untuk menyederhanakan cara PPKS mendapatkan informasi kesehatan. Kami menggabungkan teknologi dan sistem screening instan untuk membantu kamu memantau status gizi secara cepat. Inisiatif ini adalah langkah nyata dalam memberikan pelayanan prima yang adaptif terhadap perkembangan teknologi demi menciptakan generasi yang lebih sehat.
